**📊 Sales Performance Dashboard (Power BI)**

This project showcases an interactive dashboard built using Power BI to analyze sales performance across regions, categories, and products.

 **Key Features:**
KPI cards for Total Sales, Total Profit, and Profit Margin
Region-wise sales comparison
Category-wise sales distribution
Year-wise sales trend analysis
Top-performing products visualization
Interactive slicers for filtering data
**Key Insights:**
West region generates the highest sales
Technology category contributes the most revenue
Sales show a strong upward trend after 2024
Overall profit margin is approximately 12.56%
**Tools Used:**
Power BI
Excel
